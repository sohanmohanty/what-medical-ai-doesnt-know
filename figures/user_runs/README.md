# User Run Figures

This folder stores figures from custom non-canonical runs launched by the user.

Examples:

- `figures/user_runs/debug_wdbc/`
- `figures/user_runs/my_custom_run/`

These figures are separate from:

- `figures/paper/` for curated paper-facing figures
- `figures/runs/` for official tracked run-summary figures
- `figures/samples/` for built-in sample and preview runs

The point of this folder is to let you experiment without mixing your personal trial runs into the official project outputs.
