# Sample Run Figures

This folder stores figures from the built-in sample commands that come with the repo.

The main examples are:

- `run_smoke_experiment.ps1`
- `run_graph_preview.ps1`

Their outputs go into subfolders such as:

- `figures/samples/smoke_test/`
- `figures/samples/graph_preview/`

These figures are useful for quick checks and demos. They are not the main tracked paper figures.

If you run your own custom experiment, its figures should go in `figures/user_runs/` instead.
