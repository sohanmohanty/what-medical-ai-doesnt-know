# Sample Run Outputs

This folder stores outputs from the built-in sample and preview commands.

Examples:

- `run_smoke_experiment.ps1`
- `run_graph_preview.ps1`

Each sample run gets its own subfolder. A sample run folder can contain:

- `metrics.csv`
- `summary.csv`
- `predictions.csv` when enabled
- `run_manifest.json`
- `masks/` when enabled

These outputs are meant for quick checks and demonstrations, not for the official tracked project record.
