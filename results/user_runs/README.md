# User Run Outputs

This folder stores outputs from custom non-canonical runs launched by the user.

Each custom run gets its own subfolder so it does not get mixed with:

- official tracked outputs in `results/metrics/`, `results/predictions/`, `results/manifests/`, and `results/masks/`
- built-in sample outputs in `results/samples/`

A user-run folder can contain:

- `metrics.csv`
- `summary.csv`
- `predictions.csv` when enabled
- `run_manifest.json`
- `masks/` when enabled
